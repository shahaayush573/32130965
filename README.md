## Project Overview

This repository contains the practical notebook work for the Databases and Analytics assignment based on the NorthStar Urban Mobility and Logistics case study.

NorthStar is a regional organisation that manages transport, last-mile delivery, warehouse dispatch, electric vehicle charging hubs, route planning systems, and a mobile customer platform. The company is facing issues such as fragmented data, delivery delays, failed services, customer complaints, hub performance problems, vehicle maintenance issues, and mobile app reliability concerns.

The aim of this project is to use different database and analytics technologies to investigate these problems and propose a unified data-driven solution.

The project uses:

- Python data processing
- SQL within R
- R analytics and visualisation
- MongoDB Atlas NoSQL database design
- PyMongo CRUD operations
- MongoDB aggregation
- Indexing and query optimisation
